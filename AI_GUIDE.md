# AI 智慧建議功能使用指南 🤖

## ✨ 功能介紹

### 什麼是 AI 智慧建議？

每次辨識完女朋友的表情後，系統會使用 AI（GPT-3.5）生成**全新的、個性化的建議**，而不是從預設的建議庫中隨機選擇。

### 優勢

1. **每次都不同**：AI 每次都會生成新的建議，不會重複
2. **更自然**：建議更像真人在給建議，不是固定模板
3. **更貼心**：根據信心度和表情微調建議內容
4. **更有趣**：AI 的回應更幽默、更有創意

## 🔑 如何取得 OpenAI API Key

### 步驟 1：註冊 OpenAI 帳號

1. 前往 [OpenAI 官網](https://platform.openai.com/)
2. 點擊 "Sign up" 註冊帳號
3. 驗證電子郵件

### 步驟 2：建立 API Key

1. 登入後，點擊右上角頭像
2. 選擇 "View API keys"
3. 或直接前往：https://platform.openai.com/api-keys
4. 點擊 "Create new secret key"
5. 給 Key 取個名字（例如：emotion-recognition）
6. 點擊 "Create secret key"
7. **重要**：立即複製 API Key（只會顯示一次！）

### 步驟 3：儲存 API Key

⚠️ **安全提醒**：
- 不要分享你的 API Key
- 不要上傳到 GitHub
- 不要在公開場合展示

## 🚀 使用方法

### 方法 1：在應用程式中設定（推薦）

1. 開啟應用程式
2. 在左側邊欄找到 "AI 智慧建議設定"
3. 輸入你的 OpenAI API Key
4. 看到 "✅ AI 智慧建議已啟用！" 就成功了
5. 開始辨識表情，享受 AI 建議！

### 方法 2：使用環境變數

在專案目錄建立 `.env` 檔案：

```env
OPENAI_API_KEY=sk-your-api-key-here
```

## ⚙️ 技術細節

### AI 如何生成建議？

1. **輸入資訊**：
   - 辨識的表情類別（例如：餓了）
   - 表情名稱（例如：餓了 😋）
   - 信心度（例如：87.3%）

2. **AI 處理**：
   - 理解當前情境
   - 生成貼心的建議
   - 加入適當的 emoji
   - 確保語氣可愛友善

3. **輸出結果**：
   - 50-80 字的建議
   - 實用且容易執行
   - 幽默但不誇張

### 提示詞範例

系統會傳送類似這樣的提示給 AI：

```
你是一個貼心、可愛、幽默的戀愛顧問助手。

女朋友現在的表情是：餓了 😋
辨識信心度：87.3%
情境說明：她看起來餓了，想吃東西

請給出一個：
1. 非常可愛且貼心的建議（要有emoji）
2. 語氣要像是在跟好朋友聊天一樣輕鬆
3. 建議要實用且容易執行
4. 大約50-80字
5. 要幽默但不要太誇張
6. 加入一些具體的行動建議
```

## 🛡️ 安全性與隱私

### 資料傳輸

- API Key 不會被儲存在伺服器
- 只會傳送表情類別，不會傳送照片
- 所有通訊都經過 HTTPS 加密

### 隱私保護

**不會傳送到 OpenAI 的資料**：
- ✅ 照片內容
- ✅ 個人身份資訊
- ✅ 使用者姓名

**會傳送到 OpenAI 的資料**：
- ❌ 表情類別（例如：開心、生氣）
- ❌ 信心度數值

## ❓ 常見問題

### Q1: 我一定要使用 API Key 嗎？

**A**: 不用！不設定 API Key 也可以正常使用，系統會使用預設的建議清單。

### Q2: API Key 會過期嗎？

**A**: 不會自動過期，但如果帳戶餘額用完就無法使用。可以隨時充值。

### Q3: 如果 API 調用失敗會怎樣？

**A**: 系統會自動切換回預設建議，不會影響正常使用。

### Q4: 可以使用其他 AI 服務嗎？

**A**: 目前只支援 OpenAI，未來可能會加入 Google Gemini、Claude 等。

### Q5: 如何知道 API 還有多少額度？

**A**: 前往 [OpenAI Usage 頁面](https://platform.openai.com/usage) 查看。

### Q6: 建議生成需要多久？

**A**: 通常 1-3 秒，視網路速度而定。

### Q7: 可以自訂 AI 的風格嗎？

**A**: 目前使用固定的提示詞，未來版本可能開放自訂。
