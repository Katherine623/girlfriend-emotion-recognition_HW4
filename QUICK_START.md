# 🚀 快速開始指南

## 🎯 現在就能用！

### 方式一：展示模式（最快，推薦演示使用）

1. **訪問部署網址**：
   - https://girlfriend-emotion-recognition-hw4.streamlit.app/

2. **啟用展示模式**：
   - 勾選「🎭 啟用展示模式（無需訓練模型）」
   - 上傳任何照片即可體驗完整功能
   - 展示模式會模擬情緒識別結果

3. **功能體驗**：
   - ✅ 上傳照片
   - ✅ 情緒分析（模擬）
   - ✅ AI 建議
   - ✅ 歷史記錄
   - ✅ 統計圖表

**適用場景**：
- 展示應用介面和流程
- 快速體驗所有功能
- 不需要準備訓練資料

---

### 方式二：訓練真實模型（準確識別）

如果你想要真實的情緒識別功能，需要準備訓練資料並訓練模型：

#### Step 1: 準備訓練資料

**選項 A - 自動下載（推薦）**：

```bash
# 從 Unsplash 下載（無需 API key）
python download_training_data.py

# 或從 Pexels 下載（需要免費 API key）
# 先註冊 https://www.pexels.com/api/
python download_with_api.py
```

**選項 B - 手動收集**：

詳見 [TRAINING_DATA_GUIDE.md](TRAINING_DATA_GUIDE.md)

#### Step 2: 訓練模型

1. 訪問應用的「訓練模型」標籤
2. 確認資料夾結構正確（每個情緒至少 3 張照片）
3. 點擊「🚀 開始訓練模型」
4. 等待訓練完成（約 5-10 分鐘）

#### Step 3: 使用模型

訓練完成後：
- 展示模式會自動關閉
- 可以上傳照片進行真實的情緒識別
- 結果基於訓練的神經網路模型

## 🔧 技術細節

### 展示模式運作原理
- 上傳照片後生成隨機但合理的預測機率
- 一個情緒會佔主導地位（看起來更真實）
- 所有 UI 功能完全可用
- 標註「🎭 展示模式」以區分真實預測

### 真實模型架構
- 基礎模型：MobileNetV2
- 輸出層：8 個情緒類別
- 訓練方式：遷移學習 + 微調
- 圖片大小：224x224 pixels

## ❓ 常見問題

**Q: 展示模式的預測準確嗎？**
A: 不準確，是隨機模擬的，僅用於展示應用流程。

**Q: 我可以跳過訓練直接使用嗎？**
A: 可以！勾選展示模式就能立即使用所有功能。

**Q: 訓練需要多久？**
A: 本地訓練約 5-10 分鐘（取決於電腦效能）。

**Q: 訓練資料需要多少張照片？**
A: 最少每個情緒 3 張，建議每個情緒 15-30 張以獲得更好效果。

**Q: 我可以在 Streamlit Cloud 上訓練嗎？**
A: 不建議，Streamlit Cloud 資源有限且會超時。建議本地訓練後上傳模型檔案。
